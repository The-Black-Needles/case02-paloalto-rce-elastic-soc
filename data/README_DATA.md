# data/
- Coloque aqui os **logs brutos** do laboratório (preferência: **NDJSON**).
- Exemplos de nomes:
  - data/raw/palo_system.ndjson
  - data/raw/palo_traffic.ndjson
  - data/raw/palo_url.ndjson
- Se vierem em CSV, manter em `data/raw/` (vamos fornecer ingestão alternativa).
- **Não** versionar dados reais (já ignorados pelo .gitignore).
